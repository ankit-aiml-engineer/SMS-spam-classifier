# 📩 SMS Spam Classifier using NLP & Machine Learning

> 🚀 A machine learning project that classifies SMS messages as **Spam** or **Ham (Not Spam)** using Natural Language Processing techniques.

---

## 📌 Overview

This project builds an intelligent system to automatically detect spam messages from SMS data.
It uses NLP techniques and machine learning algorithms to analyze text and make accurate predictions.

---

## 🎯 Objective

* Detect spam messages automatically
* Improve message filtering systems
* Apply NLP techniques on real-world text data

---

## 📊 Dataset

The dataset contains SMS messages labeled as:

* **Ham** → Normal messages
* **Spam** → Unwanted messages

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* NLP (TF-IDF / CountVectorizer)
* Streamlit (for web app)

---

## 🔄 Workflow

1. Data Cleaning
2. Text Preprocessing
3. Feature Extraction (TF-IDF)
4. Model Training
5. Prediction System

---

## 🤖 Model

* Algorithm: **Naive Bayes / Machine Learning Model**
* Used for text classification with high accuracy

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
